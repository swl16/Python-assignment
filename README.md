# Python-assignment

* You will need the following external libraries:

    pip install tkcalendar matplotlib _(in the terminal)_


* In **PyCharm**, open Setting then click Interpreter,

    Click the "+" button at the top left then search for tkcalendar and matplotlib and install it into Interpreter


    • tkcalendar: For the date picker in the Expense Tracker
    • matplotlib: For generating pie charts in the Expense Tracker 


* Ensure your project folder contains the following files:

    • mainmenu.py 

    • ExpensesTracker.py 

    • GoalTracker.py 

    • tax.py 

    • coin.png (Icon file)
